# Eutelsat Unstructured Documents (Demo)

This folder contains Eutelsat-branded reference material used by Cortex Search in this demo. Content is synthesized for demonstration and draws on the provided reports (`DOC_Investors_Presentation-FY2024-25_EN_050825.pdf`, `DOC_Investors_Q1-2025-26-Presentation_EN-211025.pdf`, `DOC_Investors_ECOM-Consolidated-accounts-FY25_EN_101025.pdf`, `DOC_Investor_Amendment-URD-2024-25_EN_251125.pdf`, `DOC_Group_URD_Eutelsat-Sustainability-Statement-2024-25_EN_301025.pdf`, and related investor PDFs) plus public website facts (https://www.eutelsat.com/) and [Wikipedia](https://en.wikipedia.org/wiki/Eutelsat). Do not treat as production data.

## Folders
- finance/: financial highlights, funding, revenue mix
- strategy/: multi-orbit priorities, vertical plays
- network/: footprint, performance, resilience
- sales/: partner guidance across broadcast, mobility, enterprise, government
- marketing/: messaging, campaign themes
- hr/: people policy summary
